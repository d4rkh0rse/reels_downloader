## :mag_right: Sample Usage <a name="features"></a>

Run normally:

~~~
git clone https://github.com/d4rkh0rse/reels_downloader.git
cd d4rkh0rse/
pip3 install -r requirements.txt    
./reels_downloder.py
~~~
